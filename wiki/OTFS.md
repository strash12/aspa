# OTFS

## Описание
Orthogonal Time Frequency Space — метод модуляции, который модулирует символы в Delay-Doppler (DD) области вместо времени или частоты. Ключевое преимущество: DD-канал почти статичен даже при высоких скоростях, в отличие от OFDM где канал быстро меняется во времени.

## Содержание

### Процесс модуляции (TX chain)
```
Биты → QPSK → DD-сетка (M×N) → Вставка пилота → ISFFT → Heisenberg → CP/ZP → Время-сигнал
```

1. **DD-сетка → частотно-временная область (ISFFT)**: 2D обратное преобразование Фурье
2. **Частотно-временная → время (Heisenberg/OFDM)**: IFFT по поднесущим + CP/ZP
3. **Матричная модель**: `s = F_OTFS · x_dd`

### Демодуляция (RX chain)
Обратный процесс: время → частотно-временная → DD-область

### Матрица канала
`r = G · s + w` где G — матрица канала с дробными сдвигами
