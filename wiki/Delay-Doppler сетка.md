# Delay-Doppler сетка

## Описание
Двумерная сетка M×N, в которой символы OTFS модулируются по задержке (delay) и Допплеру (Doppler). Каждая ячейка — один DD-bin.

## Связи
- [[OTFS]] — модуляция в DD-области
- [[DD разрешение]] — разрешение сетки по задержке и Допплеру
- [[Fractional Doppler]] — когда путь не попадает точно в bin
- [[Guard-зона]] — нулевые bins вокруг пилота
- [[OTFS параметры]] — конкретные размеры сетки

## Содержание

### Разрешение
```
Δτ = 1/(M·Δf) — разрешение по задержке
Δν = 1/(N·T)   — разрешение по Допплеру
```

При M=64, N=32, Δf=150 кГц:
- Δτ ≈ 104 нс
- Δν ≈ 4.69 кГц

### Fractional сдвиги
Когда путь канала не попадает точно в bin DD-сетки, происходит "размазывание" по соседним бинам. Это основная проблема оценки канала в OTFS.
